Built upon [DSET]([sysulic/DSET](https://github.com/sysulic/DSET)).

### Running BQS (BQNP solver)

```shell
BQS.exe <input-qnp-file> <output-file>
```

The `<input-qnp-file>` should follow the QNP format that used by [qnp2fond](https://github.com/bonetblai/qnp2fond) and the policy will be wrote to `<output-file>`.
